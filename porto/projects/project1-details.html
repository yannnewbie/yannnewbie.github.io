Sistem Informasi Perpustakaan

Web Development | Januari 2023 - April 2023

Tentang Proyek

Sistem Informasi Perpustakaan adalah aplikasi web yang dikembangkan untuk memudahkan manajemen operasional perpustakaan sekolah. Aplikasi ini memungkinkan admin untuk mengelola koleksi buku, anggota perpustakaan, dan transaksi peminjaman.

Fitur Utama

  • Manajemen Katalog Buku (CRUD)
  • Pendaftaran Anggota Perpustakaan
  • Sistem Peminjaman dan Pengembalian Buku
  • Notifikasi Email untuk Pengingat Pengembalian
  • Laporan Statistik Peminjaman
  • Pencarian Buku dengan Filter

Teknologi yang Digunakan

HTML5 CSS3 JavaScript PHP MySQL Bootstrap

Kontribusi Saya

Saya bertanggung jawab untuk mengembangkan seluruh front-end aplikasi dan beberapa modul back-end termasuk sistem peminjaman buku dan manajemen anggota. Saya juga mendesain database dan membuat dokumentasi pengguna.
